Welcome to Spin Web

Spin is a privacy-first OSINT investigation toolkit. This is the web mini version — a lightweight, client-side toolkit that runs entirely in your browser. No data is sent to any server. All analysis happens locally using JavaScript.

For the full experience — including multi-identity browsing, AI-powered agents, real-time entity sync, investigation timelines, and advanced privacy controls — install the desktop app.
